========================================================================
       STATIC LIBRARY : BinaryString
========================================================================


We use this project to compile the binarystring files, which are actually 
unicode text files saved under the "stringbin" extension.

/////////////////////////////////////////////////////////////////////////////

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named BinaryString.pch and a precompiled types file named StdAfx.obj.
